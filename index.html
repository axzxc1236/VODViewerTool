There are two versions of Twitch API you can choose, doesn't affect the tool's functionality as of now.
  • V5
    • V5 is the old API which has been announced by Twitch as depreciated
    (For about 2 years when I am writing this text)
    That means this API will be shutdown in a unknown time
    This version of API doesn't require user authentication.
  • Helix
    • Helix is the currently official supported API.
    Requires a token to use, which requires a Twitch account
    This tool doesn't need any permission that can access your private information
    Your token stays in your browser unless you are using a malicious browser (extension)
